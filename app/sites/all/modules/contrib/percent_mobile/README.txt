-- SUMMARY --

The Percent Mobile module integrates the Percent Mobile mobile analytics code

For a full description of the module, visit the project page:
  http://drupal.org/project/percent_mobile

To submit bug reports and feature suggestions, or to track changes:
  http://drupal.org/project/issues/percent_mobile


-- REQUIREMENTS --

None.


-- INSTALLATION --

* Install as usual, see http://drupal.org/node/70151 for further information.


-- CONFIGURATION --

* Configure user permissions in Administer >> User management >> Permissions >>
  percent_mobile module:

  - administer percent mobile

    Users in roles with the "administer percent mobile" permission will be
    able to add and edit the Percent Mobile tracking code.

* Add or change the Percent Mobile tracking code Administer >> Site configuration >>
  Percent Mobile.
